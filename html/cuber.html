Cuber

About The Game

Cuber is a 2D platformer game with enemies shooting at the player. The main goal is to get to the end of levels and jump into the portal. Side goals are killing the enemies via bombs and collecting coins to score points.

Where To Play The Game

Cuber is a browser based game meaning it is played through the browser. Cuber also only supports keyboard and mouse.
